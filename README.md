rgu_research_frontend-main is the source code from main GIT branch of rgu_research_frontend repository hosted on GitHub

GitHub link - https://github.com/Hesh1998/rgu_research_frontend.git
* This is the codebase for the Frontend of the Web Application which was hosted on AWS Cloud.
* To run this code please follow below steps,
  1) Use the same AWS EC2 Instance created for the Web Backend hosting.
  2) Clone the GitHub repository.
  3) Install and configure Nginx on the EC2 Instance, and serve the Web Frontend via port 80 of the Nginx web server.
  4) Access the Web Application via the Public IPv4 address of the AWS EC2 Instance using any standard web browser.
